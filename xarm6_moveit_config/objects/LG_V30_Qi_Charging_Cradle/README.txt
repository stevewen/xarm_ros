# Summary

The LG V30 is a large phone, So large that i have had trouble finding the right spot when charging it wirelessly.
This is a wireless charging cradle for your LG V30, it keeps the phone in the perfect spot, it doesn't hold on to the phone, just keeps it loosely in place. Perfect for the bedside.

I have added a version with the V30 logo, because it looks cool, and a skeleton version to save you some plastic.

It fits a Ø70mm x 9mm Charger, i bought this one for about 3€: 
https://www.aliexpress.com/item/Qi-Wireless-Charger-Charging-Pad-Mat-for-Samsung-Galaxy-S8-Plus-S7-Google-Nexus-4-5/32822633099.html
